> Validation checklist and workflows.

**Configuration**
- Slug unique; required fields present; no `customInstructions`.
- Regex patterns compile; dry‑run search matches only intended files.
- `whenToUse` clearly distinct from other modes.

**Instructions**
- Files well‑formed; patterns consistent; examples reflect real tool syntax.

**Functional**
- Mode appears & activates; groups enforced; edge cases handled.

**Integration**
- Orchestrator routing clear; boundaries explicit; handoffs defined.
